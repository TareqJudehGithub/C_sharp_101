Parameter-less Constructor VS Parameterized Constructor
 
 Parameter-less /default Constructor
  - A constructor without parameters.
    public ClassName()
    {

    }

 Parameterized Constructor
  - A constructor with one or more parameters.
    public ClassName(param1, param2)
    {
      
    }

- It is possible to have more than one constructor for a class.
- The default constructor is always executed first.
