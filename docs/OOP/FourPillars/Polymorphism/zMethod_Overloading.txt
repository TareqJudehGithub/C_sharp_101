Method Overloading - Compile-Time Polymorphism

 - Method overloading allows a class to have a multiple methods with the same name but
   with different signature. In other words, multiple methods with the same name but with
   different parameter types and numbers.
 - Method Overloading allows developers to define multiple versions of a method, with 
   different parameter numbers or types.
 - A Method signature means the method name and its parameters.
