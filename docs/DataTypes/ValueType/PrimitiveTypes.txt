Primitive Types 
 - A type (data type) specifies what type of value we want to store in a 
   variable.
 - Primitive Types store a single value.
 - Primitive Types are basic building blocks of non-Primitive types.
  

sbyte, byte
ushort, short
uint, int
ulong, long

float
double
decimal

char
bool

  int 
  long
  float
